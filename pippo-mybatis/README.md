# Pippo - MyBatis

A _microservice_ for the Person domain with **Pippo** and **MyBatis**.

## Requirements

- JDK 1.8
- Apache Maven 3.x

## Building

The first time just use:

`$ mvn -N io.takari:maven:wrapper`

For execute use:

`$ ./mvnw clean package exec:exec`

For integration test use:

`$ ./mvnw clean verify`
